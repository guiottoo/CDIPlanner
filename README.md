### O CDIPlanner é uma aplicação web para simulação de rendimentos com base na taxa CDI, obtida via API REST.

#### Tecnologias:

- React
- Vite
- Bootstrap
- jsPDF
- jspdf-autotable

#### Funcionalidades:

- Consulta da taxa CDI via API REST
- Simulação de rendimento informando valor e prazo
- Exibição do resultado da simulação
- Geração de relatório em PDF
- Interface responsiva e amigável
